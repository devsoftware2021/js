JavaScript Objects and Arrays

1. Object Literals

Objects are collections of key-value pairs, where keys are strings and values can be any data type.

const person = { name: "John", age: 30, isMarried: true }; console.log(person.name); // Output: John console.log(person.age); // Output: 30

In the example above, we define an object called person and access its properties using dot notation (e.g., person.name).

2. Array Manipulation

Arrays are used to store multiple values in a single variable. Here are some common methods:

push()

const fruits = ["Apple", "Banana"]; fruits.push("Orange"); console.log(fruits); // Output: ["Apple", "Banana", "Orange"]

The push() method adds an item to the end of the array.

pop()

fruits.pop(); console.log(fruits); // Output: ["Apple", "Banana"]

The pop() method removes the last item from the array.

shift()

fruits.shift(); console.log(fruits); // Output: ["Banana"]

The shift() method removes the first item from the array.

unshift()

fruits.unshift("Grapes"); console.log(fruits); // Output: ["Grapes", "Banana"]

The unshift() method adds an item to the beginning of the array.

slice()

const newFruits = fruits.slice(1); console.log(newFruits); // Output: ["Banana"]

The slice() method creates a shallow copy of a portion of the array, starting from the specified index.

splice()

fruits.splice(1, 1, "Pineapple"); console.log(fruits); // Output: ["Grapes", "Pineapple"]

The splice() method changes the contents of an array by removing or replacing existing elements.

3. Iterating Over Arrays

JavaScript provides several methods to iterate over arrays:

map()

const numbers = [1, 2, 3]; const doubled = numbers.map(num => num * 2); console.log(doubled); // Output: [2, 4, 6]

The map() method creates a new array populated with the results of calling a provided function on every element in the array.

filter()

const evenNumbers = numbers.filter(num => num % 2 === 0); console.log(evenNumbers); // Output: [2]

The filter() method creates a new array with all elements that pass the test implemented by the provided function.

reduce()

const sum = numbers.reduce((accumulator, currentValue) => accumulator + currentValue, 0); console.log(sum); // Output: 6

The reduce() method applies a function against an accumulator and each element in the array (from left to right) to reduce it to a single value.

forEach()

numbers.forEach(num => console.log(num)); /* Output: 1 2 3 */

The forEach() method executes a provided function once for each array element.

4. Object Iteration

To iterate over objects, you can use:

for...in

const person = { name: "John", age: 30, isMarried: true }; for (let key in person) { console.log(key + ": " + person[key]); } /* Output: name: John age: 30 isMarried: true */

The for...in loop iterates over the enumerable properties of an object.

Object.keys()

const keys = Object.keys(person); console.log(keys); // Output: ["name", "age", "isMarried"]

The Object.keys() method returns an array of a given object's own enumerable property names.

Object.values()

const values = Object.values(person); console.log(values); // Output: ["John", 30, true]

The Object.values() method returns an array of a given object's own enumerable property values.

5. Common Errors and Invalid Options

1. Invalid Method on Undefined

let undefinedVariable; undefinedVariable.push(1); // Error: Cannot read properties of undefined (reading 'push')

If you try to call an array method like push() on an undefined variable, it will result in an error because the variable is not defined or initialized.

2. Invalid Method on Non-Array

let person = {name: "John", age: 30}; person.push("New item"); // Error: person.push is not a function

Methods like push() are only applicable to arrays. Calling them on non-array objects, such as a plain object, will throw an error.

3. Invalid Object Key Access

const person = {name: "John"}; console.log(person."name"); // Error: Unexpected string

When accessing object properties, use dot notation correctly. Accessing a property with quotes around the key, as shown above, will cause a syntax error.

4. Invalid Index in Array Methods

let fruits = ["Apple", "Banana", "Cherry"]; console.log(fruits.slice(-5)); // Error: slice index out of bounds (only -3 available)

Attempting to use an invalid index in array methods like slice() can lead to unexpected results. Negative indices are allowed, but out-of-bounds negative indices will return empty arrays.

5. for...in with Arrays

const numbers = [1, 2, 3]; for (let index in numbers) { console.log(numbers[index]); } /* Output: 1 2 3 */

Although you can iterate over arrays with for...in, it's not recommended because it also iterates over array methods and properties, which could lead to unexpected results. Use forEach() or other array methods instead.
