JavaScript DOM Manipulation

The Document Object Model (DOM) represents the structure of a web page and allows JavaScript to interact with and modify it dynamically. DOM manipulation is essential for creating interactive web applications.

1. Selecting Elements

To manipulate elements in the DOM, you first need to select them. The most common methods are:

  • document.querySelector(): Selects the first element that matches a CSS selector.
  • document.querySelectorAll(): Selects all elements that match a CSS selector.
  • document.getElementById(): Selects an element by its ID.
  • document.getElementsByClassName(): Selects elements by their class name.
  • document.getElementsByTagName(): Selects elements by their tag name.

const mainDiv = document.getElementById('main'); // Select by ID const paragraphs = document.querySelectorAll('.text'); // Select by class console.log(mainDiv); // Logs the div with id "main" console.log(paragraphs); // Logs a NodeList of paragraphs with class "text"

2. Modifying Elements

Once an element is selected, you can change its content, attributes, or styles.

Example: Changing Content and Styles

Original Message

const message = document.getElementById('message'); message.textContent = "Updated Message"; // Changes the text message.style.color = "blue"; // Changes the text color to blue message.style.fontSize = "20px"; // Changes the font size

3. Creating and Adding Elements

New elements can be dynamically created and added to the DOM using document.createElement() and methods like appendChild() or append().

Example: Creating and Appending Elements

    const list = document.getElementById('list'); const newItem = document.createElement('li'); // Create a new li element newItem.textContent = "New List Item"; // Add content to the li list.appendChild(newItem); // Append the li to the ul

    Result: The unordered list (<ul>) now contains a new list item (<li>).

    4. Removing Elements

    Elements can be removed using the remove() method or removeChild() on the parent element.

    5. Event Listeners

    You can attach event listeners to elements to make them interactive.

    Example: Adding an Event Listener

    const button = document.getElementById('clickButton'); button.addEventListener('click', () => { alert('Button clicked!'); });

    When the button is clicked, an alert message will be displayed.

    6. Summary

    • DOM manipulation is a core part of creating dynamic web applications.
    • Use querySelector and related methods to select elements efficiently.
    • Modify elements using properties like textContent, innerHTML, and style.
    • Create and append new elements dynamically with createElement().
    • Handle interactivity by attaching event listeners to elements.
    Note: While DOM manipulation is browser-specific, understanding it can help when using Node.js tools that simulate browser-like behavior.
