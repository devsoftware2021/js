JavaScript Basics

1. Syntax: Understanding Variables and Constants

In JavaScript, you can declare variables and constants using var, let, and const.

var

var is function-scoped and can be re-declared and updated. It is less commonly used now due to let and const.

var name = "John"; console.log(name); // Output: John name = "Doe"; console.log(name); // Output: Doe

let

let is block-scoped and can be updated but not re-declared in the same scope.

let age = 30; console.log(age); // Output: 30 age = 31; console.log(age); // Output: 31
// Error: Cannot re-declare a variable with let in the same scope let age = 40; // SyntaxError

const

const is block-scoped and cannot be updated or re-declared. Use it for values that should not change.

const country = "India"; console.log(country); // Output: India
// Error: Assignment to a constant variable country = "USA"; // TypeError

2. Data Types

JavaScript supports various data types, including primitive and non-primitive types.

Primitive Data Types

  • String: Used to store text.
  • Number: Used to store numbers.
  • Boolean: Represents true or false.
  • Null: Represents an empty or non-existent value.
  • Undefined: Represents a variable declared but not assigned a value.
// Strings let greeting = "Hello, World!"; console.log(greeting); // Output: Hello, World! // Numbers let score = 100; console.log(score); // Output: 100 // Boolean let isActive = true; console.log(isActive); // Output: true
// Error: Attempting to use undefined without declaration console.log(notDefined); // ReferenceError

Non-Primitive Data Types

Objects and arrays are examples of non-primitive types.

// Objects let user = { name: "John", age: 30 }; console.log(user.name); // Output: John // Arrays let colors = ["Red", "Green", "Blue"]; console.log(colors[1]); // Output: Green

3. Operators

JavaScript operators are used to perform operations on values and variables.

Arithmetic Operators

Used to perform basic mathematical operations.

let a = 10, b = 5; console.log(a + b); // Output: 15 console.log(a - b); // Output: 5 console.log(a * b); // Output: 50 console.log(a / b); // Output: 2

Assignment Operators

Used to assign values to variables.

let x = 10; x += 5; // x = x + 5 console.log(x); // Output: 15
// Error: Using const with an assignment operator const y = 10; y += 5; // TypeError: Assignment to constant variable

Comparison Operators

Used to compare two values.

console.log(10 == "10"); // Output: true (value comparison) console.log(10 === "10"); // Output: false (value and type comparison) console.log(10 > 5); // Output: true

Logical Operators

Used to combine conditions.

let isLoggedIn = true; let isAdmin = false; console.log(isLoggedIn && isAdmin); // Output: false console.log(isLoggedIn || isAdmin); // Output: true console.log(!isLoggedIn); // Output: false

Ternary Operator

Used as a shorthand for conditional expressions.

let age = 20; let status = age >= 18 ? "Adult" : "Minor"; console.log(status); // Output: Adult
